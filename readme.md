### 환경 설정 (Windows)

1. Python 가상환경 생성 및 활성화:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

> PowerShell에서 스크립트 실행이 차단될 경우, 아래 명령어를 먼저 실행하세요:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```
